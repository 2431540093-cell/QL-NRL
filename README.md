# QL-NRL (Training Day Management)
## Giới thiệu
Hệ thống quản lý ngày học (Training Day) cho sinh viên:
- Tạo sự kiện
- Đăng ký tham gia
- Điểm danh (QR)

## Công nghệ sử dụng
- Next.js (App Router)
- Prisma ORM
- PostgreSQL

## Tiến độ hiện tại
### Đã hoàn thành
- Setup project Next.js
- Kết nối PostgreSQL
- Cài đặt Prisma
- Xây dựng CRUD User API

### Đang thực hiện
- Thiết kế TrainingDay & Registration

## Cách chạy project

```bash
npm install
npm run dev
